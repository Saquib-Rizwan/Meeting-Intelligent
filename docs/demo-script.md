# Demo Script

1. Upload transcript
Show the dashboard upload panel and select a `.txt` or `.vtt` transcript.

2. Confirm processing
Point to the processing state and wait for the “Insights ready” confirmation.

3. Review extracted outputs
Open the meeting detail page and show:
- Meeting Summary Card
- Decisions
- Action Items

4. Show explainability
Scroll through the sentiment timeline and speaker sentiment chart to explain how the system exposes meeting tone and evidence.

5. Ask a question
Use the chat panel to ask a practical question such as:
`Why was the API launch delayed?`

6. Show answer quality
Highlight:
- concise summary
- key points
- supporting evidence
- confidence score
- meeting count
- transcript segment count

7. Show cross-meeting intelligence
Return to the dashboard and demonstrate recurring topics, repeated concerns, and common actions.
